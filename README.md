# big_event

vue项目建立(人生苦短，我用webstorm。)
```
npm init  vue@latest
```

加入依赖
```
npm install element-plus --save     #饿了么UI 
npm install axios                   #类似hutool的HttpUtils 就一请求工具
npm install 
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
